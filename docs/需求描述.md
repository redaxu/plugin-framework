# 项目需求描述

## 项目概述

创建一个基于 PF4J 和 Spring Boot 2.2 的插件管理与运行框架，实现插件的动态加载、启动、停止和管理功能。

## 核心需求

### 1. 插件管理模块

- 实现插件生命周期管理：加载、启动、停止、重载
- 提供插件管理 API，支持动态管理插件
- 自动扫描并加载指定目录下的插件 JAR 文件

### 2. 活动管理插件（示例插件）

- 实现完整的活动管理功能，包括：
  - 活动创建、查询、更新、删除（CRUD）
  - 支持按状态筛选活动
- 技术实现：
  - Controller 层：提供 RESTful API
  - Service 层：业务逻辑处理
  - DTO 层：数据传输对象
  - DAO 层：基于 JPA 实现数据库 CRUD 操作
  - 使用 MySQL 数据库存储数据

### 3. 技术约束

- 兼容 Spring Boot 2.2 版本
- 使用 JDK 8
- 插件依赖封装在插件 JAR 中，实现依赖隔离
- 支持插件热更新，无需重启主应用

## 项目结构

- **plugin-manager**：插件管理模块（主应用）
- **activity-plugin**：活动管理插件模块（示例插件）
- **plugins/**：插件存放目录

## 交付物

1. 插件管理主应用，提供插件管理 API
2. 活动管理插件示例，展示完整的插件开发流程
3. 插件开发规范和文档
